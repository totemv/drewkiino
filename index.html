Projects Frameworks

Whirlpool

A lightweight chat framework that runs on Socket.io, supports user profiles, usernames, dynamic texts resizes, and relative timestamping. Good for a very quick messager feature.

Atlantis

Atlantis is a debug logger framework built in Swift. I was inspired by Dave Wood's XCGLogger, but felt that it was lacking in utility and some key attributes that xcode's native print() function had. I took some of his code and added a couple of things and came up with this framework. If you liked this framework, please star this repo! It would definitely motivate me to keep adding. Also, anyone willing to contribute please let me know at andrew@totemv.com.

Ocean

A highly-scalable, boilerplate node server built from scratch.

Tide

Image downloading and processing framework, everything occurs in the background and is non-invasive to the main-thread. Supports rounding of images and fitted clipping. Perfect for user profile pictures, item images, etc. Credit goes to Gavin Bunney for his Toucan framework. Tide is simply a wrapper over Toucan's core methods with added background blocks and leak fixes.

Storm

A set of native extension functions that help make developing easy by covering micro issues such as getting the height of a text or showing an activity indicator in the middle of any view.

Pacific

A set of frameworks to help bootstrap start-app grade apps
